The app WeTunes is for the Android platform that allows us to socialise, rate and suggest music among our friends to incorporate a vibrant community feeling while listening to music.

This was made for the Appsurd Competition at Techfest 2012, IIT Bombay.

Building Wetunes
----------------
Open the this repository in Eclipse with the ADT installed and then build it!

Android 2.2 or higher SDK required.

The wetunes-server (https://github.com/sankha93/wetunes-server) project also needs to be running in order to provide the backend for this app.
